Suggested challenges

  1. Write a better ghost that follows the shortest path to the PacMan.
  2. Modify the ghosts to co-operate and co-ordinate their attack
  3. Cheat! Bypass Caja sandbox or how the game uses it to:
    • kill the PacMan directly (using javascript jedi mindtricks)
    • modify the state of PacMan
    • modify the score directly
    • to allow the ghosts to communicate with each other or with the web

Whitelisted API

  • self.getPosition() -- get your own position
  • self.randomMove() -- returns a random position for your ghost to move in.
  • self.isVunerable() -- can pacman eat you right now?
  • game.map() -- get a map of the warground
  • game.look() -- find out where everyone is
  • console.log -- useful for debugging
